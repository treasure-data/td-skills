# TD 2026 Brand Color Palette

## Primary Theme: "TD 2026 UX Palette"

| Role      | Hex       | Usage                                    |
|-----------|-----------|------------------------------------------|
| Dark 1    | `#000000` | Primary text                             |
| Light 1   | `#FFFFFF` | White backgrounds, reversed text         |
| Dark 2    | `#2D40AA` | Deep blue — headings, emphasis, dark bg  |
| Light 2   | `#F7F5F9` | Off-white lavender — subtle backgrounds  |
| Accent 1  | `#FDB893` | Warm peach — callouts, highlights        |
| Accent 2  | `#FFE2BD` | Light apricot — secondary highlights     |
| Accent 3  | `#847BF2` | Purple — charts, icons, accents          |
| Accent 4  | `#C466D4` | Magenta/orchid — charts, icons, accents  |
| Accent 5  | `#FEAFD9` | Soft pink — backgrounds, tags            |
| Accent 6  | `#8BBCFD` | Sky blue — charts, icons, accents        |
| Hyperlink | `#FA731A` | Orange — links, CTAs                     |

## Chart / Data Visualization Order

Use these accent colors in this order for multi-series charts:
1. `#847BF2` — Purple
2. `#8BBCFD` — Sky blue
3. `#FDB893` — Warm peach
4. `#C466D4` — Magenta
5. `#FEAFD9` — Soft pink
6. `#FFE2BD` — Light apricot
